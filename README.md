# (En) Communicative Virtual Environment (For Online Classes)
This is a project aimed at creating an interactive environment for online classes through Virtual Reality.
Its purpose is to encourage the creation of new virtual environments, since this niche is extremely underutilized.
Using Unity for this purpose, as it has several tools not only for VR but also for user-to-user communication.
For more information, read the project's monograph.


# (Pt) AmbienteVirtualComunicativo (Para Aulas Online)
Este é um projeto que visa criar um ambiente interativo para aulas online por meio de Realidade Virtual.
Tendo como finalidade incentivar a criação de novos ambientes virtuais, uma vez que esse nicho é extremamente subutilizado.
Utilizando para tanto o Unity, já que este tem diversas ferramentas não só para RV como também para comunicação entre usuários.
Para mais informações ler a monografia do projeto.
